Hangman Game

About our project

This project is to demonstrate our new found java skills and to help learn about software development. We are creating a classic hangman game, where a random word will be generated an the user will be able to guess characters that are in the word. Every incorrect guess adds another part to the man and the gallows. If the man is hung before the user guesses the word, they lose.
